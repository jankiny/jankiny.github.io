е®ҡдҪҚvirtualenvwrapper.shи„ҡжң¬зҡ„е®үиЈ…дҪҚзҪ®

з„¶еҗҺпјҢе°Ҷvirtualenvwrapperи„ҡжң¬зҡ„дҪҚзҪ®еҶҷе…ҘеҲ°.bashrcж–Үд»¶дёӯгҖӮзӣҙжҺҘз”Ёvimзј–иҫ‘еҷЁжү“ејҖ.bashrcж–Үд»¶пјҢз„¶еҗҺе®ҡдҪҚеҲ°ж–Үд»¶зҡ„жңҖеҗҺпјҢж·»еҠ дёҖиЎҢsourceе‘Ҫд»ӨпјҢе°Ҷи„ҡжң¬дҪҚзҪ®еҶҷе…ҘгҖӮеңЁ.bashrcж–Үд»¶дёӯпјҢеҲӣе»әеҸҳйҮҸWORKON_HOMEпјҢжҢҮе®ҡPythonзҺҜеўғзҡ„еӯҳеӮЁдҪҚзҪ®пјҢеҰӮдёӢжүҖзӨәгҖӮ

е®үиЈ…NеҚЎй©ұеҠЁ

зЎ®и®ӨPythonе’ҢpipжҳҜ64bitпјҢе№¶дё”еӨ„зҗҶеҷЁжһ¶жһ„жҳҜx86_64пјҲжҲ–з§°дҪңx64гҖҒIntel 64гҖҒAMD64пјүжһ¶жһ„

жЈҖжөӢNVIDIAеӣҫеҪўеҚЎе’ҢжҺЁиҚҗзҡ„й©ұеҠЁзЁӢеәҸзҡ„жЁЎеһӢпјҢеңЁз»Ҳз«Ҝиҫ“е…Ҙпјҡ
