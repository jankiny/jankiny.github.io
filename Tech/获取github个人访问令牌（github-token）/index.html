如何获取GitHub个人访问令牌（Token）

AI导读：本文介绍了如何获取GitHub个人访问令牌（Token）。用户可以通过Fine-grained Personal Access Tokens页面或进入头像设置中的Developer Settings来生成新的token。填写所需信息时，需要选择合适的仓库权限，特别是将Content项设置为读写权限，以便能够向仓库推送内容。获得的Token可直接使用，不必特别保存，但要小心防止泄露。在管理页面上，用户还可以重新生成token或调整权限。此外，文中提及了一个实际案例，即使用GitHub token通过GitHub Action部署Hexo博客，涉及对两个仓库的访问与修改。

从Fine-grained Personal Access Tokens (github.com)或从头像 》Settings 》 Developer Settings进入。选择Generate new token，填写token的信息，选择token具有权限的仓库。

在权限这里我们主要是需要向仓库push的权限，所以将Content这个项改为读写权限。

得到的Token直接在需要的地方使用，不需要特别的保存，避免泄露个人Token。在管理页面可以重新生成，也可以增减权限。
