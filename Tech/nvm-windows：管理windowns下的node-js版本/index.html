nvm-windows：管理Windowns下的Node.js版本

使用nvm工具管理Node.js版本

AI导读：本文介绍了使用nvm-windows工具管理Node.js版本的重要性及基本操作。由于不同项目对Node.js版本有不同需求，nvm（Node Version Manager）为用户提供了方便的版本切换功能。文章首先介绍了nvm-windows的安装步骤，并提供了测试安装成功的方法。接着列出了常用指令，包括安装、卸载和查看当前可用Node.js版本等，帮助用户高效管理Node.js环境。在此之前，如果用户曾经安装过Node.js，需要先进行完全移除。

前言

由于Node.js版本更新快速，不同的项目可能需要使用不同Node.js版本的需求，因此出现了Node版本管理工具，用于在不同的Node.js版本之间进行切换。

NVM (Node Version Manager ) 是 Node.js 版本管理工具，可用于管理和切换到不同版本的 Node.js。本文要介绍的是 Windows 版本 Node.js 版本管理工具 nvm-windows 也是微软所推荐的工具。

nvm-windows 常用指令整理

  • nvm install node-version — 安裝指定版本
  • nvm uninstall node-version — 移除指定版本
  • nvm list — 查看機器中目前所安裝的版本
  • nvm use node-version — 使用指定版本
  • nvm list available — 查看當前 Node.js 版本

★ 如曾安裝過 Node.js，需在安裝 nvm-windows 前先移除完畢，移除方法。
